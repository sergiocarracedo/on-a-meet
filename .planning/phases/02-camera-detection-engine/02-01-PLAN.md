<action>
Create `internal/detector/v4l2_linux.go` with `//go:build linux` build tag.

Define the V4L2 ioctl request code and capability struct:

```go
//go:build linux

package detector

import (
    "fmt"
    "os"
    "path/filepath"
    "syscall"
    "time"
    "unsafe"

    "golang.org/x/sys/unix"
)

// V4L2 ioctl request codes — _IOR('V', 0, struct v4l2_capability)
// Uses Linux ioctl encoding: dir(bits 30-31), size(bits 16-29), type(bits 8-15), nr(bits 0-7)
var vidiocQueryCap = _IOR('V', 0, unsafe.Sizeof(v4l2Capability{}))

// _IOR macro: direction=read(2), type, nr, size → request code
func _IOR(t byte, nr byte, size uintptr) uint32 {
    return (2 << 30) | (uint32(size) << 16) | (uint32(t) << 8) | uint32(nr)
}

type v4l2Capability struct {
    Driver       [16]byte
    Card         [32]byte
    BusInfo      [32]byte
    Version      uint32
    Capabilities uint32
    DeviceCaps   uint32
    Reserved     [3]uint32
}

const (
    v4l2CapVideoCapture = 0x00000001
)
```

Implement the `V4L2Detector` struct:

```go
type V4L2Detector struct{}

func NewV4L2Detector() *V4L2Detector {
    return &V4L2Detector{}
}
```

Implement `ListDevices()`:
1. Glob pattern `/dev/video*` using `filepath.Glob`
2. For each path, call `openAndQueryCap` to check if it's V4L2_CAP_VIDEO_CAPTURE
3. Collect matching `DeviceInfo` entries with Driver, Card, Bus info from v4l2_capability
4. Return the list (or empty list with nil error if no devices found)

Helper `openAndQueryCap(path string) (DeviceInfo, bool, error)`:
1. Open device with O_RDWR using `unix.Open(path, unix.O_RDWR, 0)`
2. If EACCES, try O_RDONLY: `unix.Open(path, unix.O_RDONLY, 0)`
3. Call `unix.Ioctl(fd, vidiocQueryCap, unsafe.Pointer(&cap))` to get capabilities
4. Check `cap.Capabilities & v4l2CapVideoCapture` to filter
5. Convert driver/card/bus_info byte arrays to strings (null-terminated)
6. Return DeviceInfo
7. Close fd and return

Implement `Detect(devicePath string) (DeviceStatus, error)`:
1. Open with O_RDWR using `unix.Open(devicePath, unix.O_RDWR, 0)`
2. If success → close with `unix.Close(fd)`, return DeviceStatus{On: false, CheckedAt: time.Now()}
3. If errno is EBUSY → return DeviceStatus{On: true, CheckedAt: time.Now()}
4. If errno is EACCES → try O_RDONLY fallback:
   - Open with O_RDONLY
   - If success → close, return DeviceStatus{On: false, CheckedAt: time.Now()} with permission note
   - If fails → return error with permission hint
5. For other errors (ENOENT, ENODEV), return error

Important: Use `unix.Open`, `unix.Close`, `unix.Ioctl`, not `os.OpenFile` or `syscall.Open`. Check errno after open using `unix.EBUSY`, `unix.EACCES`, `unix.ENOENT`.
