# Plan 05-01: lsof Backend + Factory

<objective>
Build the lsof detection backend as a second Detector implementation and a factory function for runtime backend selection. The lsof backend runs `lsof /dev/videoN`, checks exit code (0=ON, 1=OFF), and falls back to V4L2 glob+VIDIOC_QUERYCAP for device enumeration (lsof cannot enumerate). The factory `detector.New("v4l2"|"lsof")` returns the appropriate backend, enabling runtime switching without changing call sites.
</objective>

Create `internal/detector/lsof_linux.go`:

```go
//go:build linux

package detector

import (
	"fmt"
	"os/exec"
	"path/filepath"
	"strings"
	"time"
)

type LsofDetector struct{}

func NewLsofDetector() *LsofDetector {
	return &LsofDetector{}
}

func (d *LsofDetector) ListDevices() ([]DeviceInfo, error) {
	paths, err := filepath.Glob("/dev/video*")
	if err != nil {
		return nil, fmt.Errorf("enumerate video devices: %w", err)
	}

	var devices []DeviceInfo
	for _, path := range paths {
		info, ok, err := openAndQueryCap(path)
		if err != nil {
			continue
		}
		if ok {
			devices = append(devices, info)
		}
	}
	return devices, nil
}

func (d *LsofDetector) Detect(devicePath string) (DeviceStatus, error) {
	cmd := exec.Command("lsof", devicePath)
	err := cmd.Run()

	if err == nil {
		return DeviceStatus{On: true, CheckedAt: time.Now()}, nil
	}

	if exitErr, ok := err.(*exec.ExitError); ok {
		if exitErr.ExitCode() == 1 {
			return DeviceStatus{On: false, CheckedAt: time.Now()}, nil
		}
	}

	return DeviceStatus{}, fmt.Errorf("lsof detection failed for %s: %w", devicePath, err)
}
```

Key points:
- ListDevices() reuses the existing `openAndQueryCap()` from v4l2_linux.go (same package, same build tag)
- Detect() runs `lsof /dev/videoN` — exit 0 = process has device open = ON
- Exit code 1 from lsof = no process holds it = OFF
- Other errors (lsof not found, permission denied) are returned as errors

```go
package detector

import "fmt"

// New creates a Detector backend by name.
// Supported methods: "v4l2", "lsof".
func New(method string) (Detector, error) {
	switch method {
	case "v4l2":
		return NewV4L2Detector(), nil
	case "lsof":
		return NewLsofDetector(), nil
	default:
		return nil, fmt.Errorf("unknown detection method: %q (supported: v4l2, lsof)", method)
	}
}
```
