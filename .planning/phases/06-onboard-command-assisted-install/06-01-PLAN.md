### Task 2: Create cmd/onboard.go with interactive wizard

**Files:**
- `cmd/onboard.go`

**Action:**
Create a new `onboard` cobra subcommand with the following flags:
- `--dry-run` (bool, default false) — preview config without sudo re-exec

The RunE flow:
1. List devices via `detector.New("v4l2").ListDevices()` → show device info
2. Use `huh.NewMultiSelect[string]` for camera selection (options: each device path + "all cameras" as default selected). Store selected paths in a `[]string`.
3. Use `huh.NewNote` or `output.Info` to explain V4L2 (direct syscall, Linux-only, faster) vs lsof (cross-platform, needs lsof installed, fallback).
4. Use `huh.NewSelect[string]("v4l2", "lsof")` for detection method.
5. Test the chosen method:
   - Create detector via `detector.New(method)`
   - `output.Warning.Printfln("Please enable your camera, then press Enter")`
   - Wait for Enter (use bufio.Scanner), run Detect(), print result
   - `output.Warning.Printfln("Please disable your camera, then press Enter")`
   - Wait for Enter, run Detect(), print result
   - If either detection fails, offer `huh.NewConfirm("Test failed. Try the other method?")` → back to step 4
6. Use `huh.NewInput().Title("Debounce").Value(&debounceStr).Placeholder("2")` — default 2.
7. Use `huh.NewInput().Title("Interval").Value(&intervalStr).Placeholder("1s")` — default "1s".
8. If `--dry-run`: print the config YAML to stdout and exit.
9. If not dry-run: write config to `huh.NewForm` confirmation group showing all selections. On confirm, write collected answers to `/tmp/on-a-meet-onboard.json` as JSON, then print sudo re-exec instructions.

Import structure:
```go
package cmd

import (
    "encoding/json"
    "fmt"
    "os"
    "time"
    "bufio"

    "github.com/charmbracelet/huh"
    "github.com/spf13/cobra"

    "github.com/sergiocarracedo/on-a-meet/internal/detector"
    "github.com/sergiocarracedo/on-a-meet/internal/output"
)
```

Use `huh.NewForm(huh.NewGroup(...fields...)).WithHeight(10)` for the form. Validate interval is parseable by `time.ParseDuration`.
