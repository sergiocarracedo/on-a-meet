# Plan 07-01: Release Automation

**Wave:** 1 (single wave)
**Objective:** Set up GoReleaser cross-platform builds with GitHub Releases on tag push.

## Tasks

1. Configure `.goreleaser.yaml` with linux/darwin + amd64/arm64 targets, CGO_ENABLED=0, ldflags, checksums
2. Create `.github/workflows/release.yml` — triggers on tag push `v*`, runs GoReleaser

## Success Criteria

- `git tag v1.0.0 && git push origin v1.0.0` triggers a GitHub Release with cross-platform binaries
