# Plan 07-01: Release Automation & Publishing

**Wave:** 1 (single wave — no dependencies)
**Objective:** Set up automated release pipeline: GoReleaser (build+GitHub Releases), Homebrew tap formula, npm wrapper package, and CI/CD workflow.

## Tasks

1. Configure `.goreleaser.yaml` (from Phase 5) with linux/darwin + amd64/arm64 targets, CGO_ENABLED=0, ldflags, checksums, and commented-out Homebrew `brews` section
2. Create `.github/workflows/release.yml` — triggers on tag push `v*`, runs GoReleaser + npm publish
3. Create `packages/npm/` with:
   - `package.json` (name, version, bin entry, postinstall script)
   - `bin/on-a-meet.js` (shebang + spawn downloaded binary)
   - `scripts/postinstall.js` (download correct platform binary from GitHub Release, extract to vendor/)
4. Document manual steps in `07-CONTEXT.md` and `07-MANUAL-STEPS.md`

## Success Criteria

- `git tag v1.0.0 && git push origin v1.0.0` triggers a GitHub Release with cross-platform binaries
- npm package can be published from `packages/npm/`
- Homebrew installation workflow documented
